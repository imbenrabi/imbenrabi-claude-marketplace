# Scene Composer

Rules for building individual scenes with cinematic structure, sensory grounding, and subtext.

**Step 1: Load reference file**

Read `${CLAUDE_SKILL_DIR}/references/scene-patterns.md` before composing any scene. It contains Hebrew opening templates, tension patterns, and closing techniques.

---

## 1. Entry: In Media Res

**Every scene begins in the middle of something.**

The first sentence must be action, sensation, or mid-dialogue. Never:
- Context-setting: "בבוקר של יום שלישי..."
- Character introduction: "עמדה לאה ליד החלון ואז..."
- Weather opening: "ירד גשם כבד..."

Instead:
- Start with the verb: "פתחה את הדלת לפני שהספיק להגיד שלום."
- Start in dialogue: "—את בטוחה?" שאל בלי להסתכל עליה.
- Start with sensation: "ריח הסיגריות דבק בבגדים גם אחרי שיצאה."
- Start after: the reader enters a moment already in progress.

## 2. Sensory Anchoring

**Ground the reader physically before any emotion or inner thought.**

Order of scene elements:
1. Sensory detail (smell, sound, texture, temperature, light)
2. Physical setting (only what is relevant to the scene's action)
3. Character action
4. Interiority or emotion (if at all)

Avoid: opening with a character's emotional state. Start with what their senses encounter, and let emotional state emerge from behavior.

## 3. Subtext Over Exposition

**Dialogue implies. Narration does not explain what dialogue already implies.**

If a character says something, and the narrator then tells us what it meant — delete the narrator's explanation.

The test: remove the explanatory sentence. Does the reader lose information they cannot recover from context? If yes, keep it. If no (because the dialogue already implies it), delete it.

Subtext lives in:
- What a character does not answer
- What they do instead of what they're asked
- The gap between what they say and what we see them doing
- Silence as a beat — noted by the narrator without explanation

## 4. Pacing via Sentence Length

**Sentence length is the primary pacing instrument.**

| Situation | Sentence Length |
|-----------|----------------|
| Tension building | Short (3–7 words) |
| Action peak | Shortest (1–4 words) |
| Interiority / memory | Long (20–40 words) |
| Description | Medium-long |
| Scene close | Variable — end on an image, not a summary |

Scene peaks use the writer's shortest sentences. A climactic moment written in 30-word sentences is a pacing failure.

Scene close: open image, interrupted action, or recontextualized detail. Never a resolution or summary sentence. The scene ends before the reader is ready for it to end.

## 5. Cinematic Blocking

**Characters exist in space. Movement carries meaning. Silence is a beat.**

Before writing a scene, know:
- Where each character is (spatial relationship)
- What they are doing with their hands/body
- What the camera would see (not what only an omniscient narrator can know)

Silence is not absence. Write it as a beat: "לא ענה." Full stop. Not "הוא לא ענה כי הוא לא ידע מה לומר." Just: "לא ענה."

Movement means something. A character who walks to the window instead of answering is communicating. Don't explain — show the movement and let the reader interpret.

## Application

Apply these rules in order:
1. Identify scene entry — revise if it begins with context-setting
2. Check first paragraph for sensory anchor
3. Scan for explanatory narration after dialogue — delete
4. Read aloud (silently) for rhythm variation — break uniform meter
5. Verify characters have spatial positions
6. Check scene end — if it closes cleanly, cut back one beat

Reference `scene-patterns.md` for working Hebrew examples of each technique.

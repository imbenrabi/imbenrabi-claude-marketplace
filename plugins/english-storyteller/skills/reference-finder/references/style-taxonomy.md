# Style Taxonomy

Controlled vocabulary for matching literary works to project signals. Use this when
constructing search queries and evaluating candidate works.

---

## Genre Tags

| Tag | What it covers |
|-----|---------------|
| `literary-fiction` | Character-driven; prose craft primary; plot secondary |
| `war-fiction` | Combat, its aftermath, military context |
| `psychological-fiction` | Interior, obsession, unreliable narrator, mind as subject |
| `domestic-fiction` | Home, family, ordinary life as battleground |
| `crime-literary` | Crime as premise; literary execution |
| `historical-literary` | Period setting; research worn as texture not information |
| `southern-gothic` | American South; grotesque; grace through violence |
| `urban-fiction` | City as character; class; social texture |
| `coming-of-age` | Formation narrative; the self under construction |

---

## Tone Descriptors

| Descriptor | What it means in literary prose terms |
|---|---|
| `spare` | Carver/Hemingway: declarative sentences; nothing unnecessary; maximum submerged |
| `lyrical` | Robinson/Morrison: earned abstraction; rhythm as meaning; language that rises |
| `fractured` | Johnson/McCarthy: time unreliable; syntax breaks under pressure |
| `ironic` | DeLillo/O'Connor: gap between what is described and how it is described |
| `brutal` | McCarthy: violence without moral comment; the world does not explain itself |
| `melancholic` | Munro/Strout: the weight of what cannot be changed |
| `precise` | Highsmith/Bolaño: every detail load-bearing; nothing decorative |
| `accumulative` | O'Brien: lists as emotional weight; cataloguing as mourning |
| `hallucinatory` | Johnson: reality unstable; the narrator may be wrong about what happened |
| `controlled` | Ishiguro/Tóibín: enormous feeling barely contained in formal prose |

## Technique Taxonomy

The bridge between a tone descriptor and a craft technique. Use these when annotating excerpts.

| Technique | What it is | Associated authors |
|---|---|---|
| `declarative_suppression` | Using plain declarative sentences to say difficult things without commentary | O'Brien, Hemingway |
| `accumulation_as_weight` | Listing objects or facts as a way of holding something too large to name | O'Brien, Sebald |
| `free_indirect_discourse` | Narrator adopts character's idiom/syntax without quotation marks | Austen, Woolf, Flaubert |
| `time_dislocation` | Past intrudes on present through specific sensory flash, not transition | Powers, O'Brien |
| `violence_without_comment` | The violent event is described at the same register as the mundane | McCarthy, Johnson |
| `iceberg_theory` | The story the reader knows is held submerged; only the tip is narrated | Carver, Hemingway |
| `body_as_compass` | When everything else is unreliable, the body's physical fact is the only truth | Johnson, Marlantes |
| `earned_abstraction` | An abstract statement that is justified by the preceding concrete moment | Robinson, Morrison |
| `dialogue_as_negotiation` | Every exchange is about power, not information | Price, Pelecanos |
| `controlled_grief` | Enormous emotion expressed through understatement and formal constraint | Ishiguro, Tóibín |

## Signal-to-Author Matrix

A quick reference for which authors to investigate given a combination of signals.

| Genre + Tone combination | Start with |
|---|---|
| War + spare + trauma | O'Brien, Powers, Klay, Marlantes |
| Psychological + fractured + dark | Johnson, McCarthy, Highsmith |
| Domestic + controlled + grief | Munro, Ishiguro, Strout |
| Literary + lyrical + spiritual | Robinson, Morrison, Tóibín |
| Urban + dialogue-heavy + class | Price, Pelecanos, Ellroy |
| Southern + gothic + violent | O'Connor, McCullers, McCarthy (early) |
| Historical + precise + period | Mantel, Tóibín, Burgess |
